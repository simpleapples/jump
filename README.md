jump
==========

CloudJump - 一个使用javascript实现的类似doodlejump的小游戏。仅支持移动设备。

使用方法
==========
使用iphone或ipad等设备打开页面后，游戏即刻开始，使用设备所有晃动，控制小人踩上云彩，向上跳。暂不支持设备横屏使用。
